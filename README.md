# 🛸 ARCHIVES VI

> **Truth is hidden in Leonida.**

ARCHIVES VI is a community-driven website dedicated exclusively to **Grand Theft Auto VI mysteries, easter eggs, theories and discoveries**.

Inspired by classic communities such as GameVicio, GTA Forums and The X-Files, the project aims to create a central archive where players can investigate, discuss and document everything hidden inside Leonida.
